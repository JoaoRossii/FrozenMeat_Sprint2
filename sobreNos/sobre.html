    Quem somos?

    Auxiliamos você a monitorar a temperatura desde o transporte da carne até dentro do seu frogorífico de uma maneira eficaz para o seu estabelecimento.

    Nossos Objetivos

    Implementação do sensor de temperatura nos frigoríficos.

    Coleta de dados sobre temperatura nos frigoríficos.

    Disponibilizar os dados sobre a temperatura para o cliente.

    Disponibilizar uma interface gráfica com os dados coletados.
